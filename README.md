# 🎵 Moodify

Moodify is an AI-based Personality Music Recommendation System built using Python and Streamlit.

## Features

- Interactive personality quiz
- Rule-based AI personality prediction
- Weighted scoring mechanism
- Personality classification
- Music recommendations
- User-friendly interface

## Technologies Used

- Python
- Streamlit
- Matplotlib
- Spotify API

## How It Works

1. User answers personality questions.
2. Responses are mapped to personality traits.
3. Scores are calculated using rule-based logic.
4. Dominant personality is identified.
5. Music recommendations are displayed.

- Machine Learning integration
